# NURLog
A Teensy 3.6-based datalogger for EV.Two - a FSAE Electric Race Car built by NU Racing and the University of Newcastle (Australia)


# Required libraries
 - [FlexCAN](https://github.com/pawelsky/FlexCAN_Library)
 - tm1638
